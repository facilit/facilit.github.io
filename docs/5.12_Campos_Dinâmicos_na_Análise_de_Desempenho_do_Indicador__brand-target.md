Registrar a análise de desempenho de um indicador é uma prática essencial no monitoramento de metas públicas, mas nem sempre os campos padrão de texto são suficientes para capturar todas as informações relevantes. Equipes de gestão frequentemente precisam registrar dados estruturados junto à análise: status de execução, responsável técnico, categorias de causa, entre outros.

Com a **Atribuição de Campos Dinâmicos à Análise de Desempenho do Indicador**, a Plataforma permite vincular campos personalizados à análise padrão de cada indicador. Dessa forma, além do texto da análise, o servidor pode preencher campos como listas de seleção, caixas de texto, responsáveis e outros tipos, diretamente na tela de análise de desempenho.

O resultado é uma análise de desempenho mais completa e estruturada, que pode ser consultada via API e exportada com precisão sem depender apenas de texto livre para registrar informações críticas da gestão.

# Campos Dinâmicos na Análise de Desempenho do Indicador

💡 **Disponibilidade:** Os campos dinâmicos na Análise de Desempenho estão disponíveis para a **Análise Padrão** dos indicadores. A configuração dos campos é realizada por administradores no módulo de Campos Dinâmicos. A posição de exibição pode ser configurada como **Acima do texto** ou **Abaixo do texto** da análise.

## Como acessar

1. Acesse o **Indicador Desejado**
2. Acesse a aba **Análise de Desempenho** 
4. Os campos dinâmicos configurados aparecerão na seção de análise, na posição definida pelo administrador

## Passo 1 — Configure os campos dinâmicos para a Análise de Desempenho (Administradores)

Antes de usar, um administrador precisa vincular os campos dinâmicos ao tipo de análise de desempenho **Padrão**.

1. Acesse **Cadastros Auxiliares** → **Campos Dinâmicos**
2. Selecione ou crie o campo dinâmico desejado
3. No campo **Posição de exibição**, selecione a opções **Indicador: Modal Performance**
4. Configure se o campo é **obrigatório** ou opcional
5. Salve as alterações — o campo estará disponível na próxima abertura da análise

### Tipos de campos dinâmicos suportados

- Caixa de única seleção (Formulário)

- Caixa de texto
- Caixa de opção
- Caixa de texto grande
- Caixa de busca de instituição
- Caixa de busca de responsável

## Passo 2 — Preencha os campos dinâmicos na Análise de Desempenho

Com os campos configurados, o servidor pode preencher as informações diretamente na tela de análise:

1. Acesse o indicador e abra a **Análise de Desempenho** do indicador
2. Preencha os campos dinâmicos exibidos (acima ou abaixo do texto)
3. Preencha o texto da análise normalmente
4. Clique em **Salvar**

## Passo 3 — Consulte os campos via API (para integrações)

Os campos dinâmicos atribuídos à análise de desempenho são retornados na **API REST** com a estrutura completa, permitindo integração com outros sistemas de gestão.

A estrutura retornada inclui:

- Valor atual da análise
- Lista de campos dinâmicos com seus valores
- Data e autoria da análise
- Status de visibilidade

**Endpoints disponíveis:**

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | `/indicators/{id}/performance-analisys` | Lista as análises do indicador com campos dinâmicos |
| POST | `/indicators/{id}/performance-analisys/{analisysId}` | Cria uma nova análise com campos dinâmicos |
| PUT | `/indicators/{id}/performance-analisys/{analisysId}` | Atualiza uma análise existente |

> ⚠️ **Atenção:** O campo `measurementdate` informado no POST/PUT deve corresponder a uma data de medição já existente no indicador. Caso contrário, o sistema retornará um erro de validação.

## Dicas para melhores resultados

✅ **Defina campos obrigatórios estrategicamente:** Tornar campos como "Status de execução" ou "Responsável técnico" obrigatórios garante que análises incompletas não sejam salvas — importante para relatórios gerenciais.

✅ **Use o tipo Formulário para categorias padronizadas:** O tipo "Caixa de única seleção (Formulário)" permite criar listas de opções padronizadas — ideal para classificar causas de desvio ou categorias de análise conforme metodologias como FCA ou 5W2H.

✅ **Alinhe a posição com o fluxo de preenchimento:** Se os campos dinâmicos são informações que contextualizam a análise, coloque-os **acima do texto**. Se são conclusões ou encaminhamentos, **abaixo do texto** faz mais sentido.

✅ **Consulte via API para painéis gerenciais:** A disponibilização dos campos via API permite que equipes de BI construam painéis automatizados que combinam texto de análise com dados estruturados — potencializando o uso das análises de desempenho.

---

## Conclusão

A atribuição de campos dinâmicos à Análise de Desempenho do Indicador transforma um registro de texto livre em um formulário estruturado e auditável — essencial para a qualidade da informação em relatórios gerenciais e integrações com outros sistemas. Com essa funcionalidade, órgãos públicos ganham padronização e rastreabilidade nas análises de indicadores, elevando o nível da gestão orientada a resultados.
