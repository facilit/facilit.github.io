# Gerador de Relatórios e Apresentações

<p style="text-align: justify;">Precisa apresentar o andamento dos projetos para a diretoria? Ou enviar um relatório consolidado para a equipe? O <strong>Gerador de Relatórios e Apresentações</strong> reúne os dados de múltiplos projetos em um único documento, no formato que você precisar: PDF, Excel, PowerPoint por template e muito mais. Você escolhe os projetos, define os campos e gera em segundos.</p>

## Como acessar

Acesse o **Acesso Rápido** na barra de navegação e procure pela opção **Gerador de Relatórios e Apresentações** (pode aparecer também como **Resumo Executivo**, dependendo da configuração do ambiente).

---

## 1. Seleção dos projetos

Ao abrir a funcionalidade, você define quais projetos comporão o relatório.

### Escolha o nível

Use o seletor de nível para indicar em qual nível hierárquico estão os projetos que deseja incluir (normalmente o nível onde ficam os projetos estratégicos do ambiente).

### Selecione os projetos

Localize os projetos usando:

- **Busca por nome:** pesquise diretamente pelo nome do projeto.
- **Filtros:** refine a listagem por **Instituição**, **Status**, **Responsável** ou **Tag**.

Marque os projetos desejados. Os itens selecionados aparecem no painel direito da tela.

### Defina a ordem

Arraste os projetos no painel direito para definir a ordem em que aparecerão no relatório.

Clique em **Próximo passo** quando a seleção estiver concluída.

---

## 2. Escolha do formato

A plataforma oferece múltiplos formatos de saída, divididos em dois grupos: **formatos diretos** (campos configuráveis) e **formatos por modelo** (baseados em template).

---

### Formatos diretos

Nesses formatos, você escolhe quais campos incluir, em qual ordem e com quais opções. O painel da esquerda lista todos os campos disponíveis; o painel da direita mostra os campos que constarão no relatório.

- **Adicionar campo:** clique no campo na lista da esquerda para incluí-lo no relatório.
- **Remover campo:** clique no campo no painel da direita para retirá-lo.
- **Reordenar:** arraste os campos no painel da direita para alterar a ordem de exibição.
- **Opções de campo:** alguns campos exibem uma seta de configuração. Por exemplo, o campo **Responsável** permite escolher entre exibir apenas o nome ou nome e e-mail.

Após configurar os campos, clique em **Próximo passo** e depois em **Gerar**.

| Formato | Características |
|---|---|
| **PDF** | Uma página por projeto; layout estruturado e pronto para impressão ou envio |
| **Word / Open Document** | Documento editável; mesma lógica de campos do PDF |
| **Excel** | Layout horizontal (uma coluna por campo); ideal para análise de dados |
| **CSV** | Texto separado por vírgulas; compatível com qualquer planilha ou sistema |
| **JSON / TXT** | Formatos estruturados para integração ou processamento de dados |

---

### Formatos por modelo (template)

Nesses formatos, você fornece um arquivo de template previamente preparado e o sistema preenche as variáveis automaticamente para cada projeto.

#### PPT por modelo

Crie um arquivo PowerPoint com a estrutura visual desejada e insira **variáveis** nos locais onde os dados de cada projeto devem aparecer. As variáveis seguem o formato `{{NOME_DA_VARIÁVEL}}`.

**Slide fixo x slide repetido por projeto**

- **Slide fixo:** conteúdo que aparece uma única vez na apresentação (ex.: capa, sumário). Basta inserir as variáveis desejadas normalmente.
- **Slide repetido:** slide que se replica para cada projeto. Nas **Notas do Apresentador** desse slide, escreva:

```
!repetir
```

O sistema interpreta essa marcação e gera uma cópia do slide para cada projeto, substituindo as variáveis pelos dados correspondentes.

**Foto do responsável**

Para exibir a foto do responsável pelo projeto, insira um objeto de imagem no slide e defina seu **texto alternativo** (Alt Text) como:

```
RESPONSÁVEL_FOTO
```

**Imagens do projeto**

Para exibir as imagens cadastradas no projeto, insira um objeto de imagem e defina seu texto alternativo como:

```
ENTIDADE_IMAGEM
```

Esse slide também deve ter `!repetir` nas notas para que a imagem seja repetida por projeto.

**Variáveis disponíveis**

A lista completa de variáveis (datas, responsável, status, campos dinâmicos e outros) está exibida na própria tela do gerador, na seção de variáveis. Consulte-a para montar o template com precisão.

Após preparar o template, faça o upload do arquivo na área indicada, clique em **Próximo passo** e depois em **Gerar PPT por Modelo**.

#### Word por modelo

Segue a mesma lógica do PPT por modelo, mas gera **um arquivo Word por projeto**. Útil para documentos como:

- Termo de abertura de projeto
- Termo de monitoramento
- Relatório individual padronizado

Prepare o template Word com as variáveis desejadas, faça o upload e gere os arquivos.

---

## Dicas

✅ **Invista no template uma única vez:** após validado, o modelo pode ser reutilizado para gerar relatórios e apresentações atualizados em segundos.

✅ **Use os filtros na seleção:** organizar e filtrar os projetos antes de marcar evita seleções erradas e agiliza a montagem do relatório.

✅ **A ordem importa:** a sequência dos projetos no painel direito determina a ordem de aparição no documento final.

✅ **Consulte as variáveis disponíveis:** a tela do gerador exibe todas as variáveis suportadas, incluindo campos dinâmicos.
