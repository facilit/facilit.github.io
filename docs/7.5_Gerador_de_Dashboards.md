# Gerador de Dashboards

<p style="text-align: justify;">Acompanhar projetos em reuniões fica muito mais fácil com um painel visual atualizado. O <strong>Gerenciador de Dashboards</strong> permite criar painéis personalizados com gráficos, tabelas e filtros em poucos cliques, sem precisar de configurações técnicas. Monte uma vez, use sempre que precisar, com dados sempre atualizados.</p>

## Como acessar

Acesse o **Acesso Rápido** na barra de navegação e procure pela opção **Gerenciador de Dashboard**.

---

## Criando um novo dashboard

1. Clique no botão **+** para criar um novo dashboard.
2. Informe o **nome** e, opcionalmente, uma **descrição**.
3. Clique em **Criar**.

O dashboard é criado vazio, pronto para receber componentes.

---

## Adicionando componentes

Clique no botão **Adicionar** na parte superior da tela para inserir um novo componente. Os tipos disponíveis são:

| Componente | O que faz |
|---|---|
| **Título** | Texto de cabeçalho para organizar visualmente o dashboard |
| **Filtros** | Painel de filtros interativos (responsável, tag, etc.) aplicáveis aos demais componentes |
| **Tabela** | Lista de entidades com colunas configuráveis |
| **Número grande** | Exibe um valor numérico em destaque (ex.: total de projetos) |
| **Gráfico de rosca** | Gráfico circular segmentado por uma dimensão (ex.: status) |
| **Barra horizontal** | Gráfico de barras horizontais segmentado por uma dimensão |

### Configurando cada componente

Ao adicionar um componente, você pode definir:

- **Título** do componente.
- **Fonte de dados:** atualmente sempre as entidades da plataforma.
- **Nível:** selecione em qual nível hierárquico buscar os dados (ex.: nível 2 para projetos).
- **Dimensão:** para gráficos, escolha como os dados serão agrupados (ex.: por status, por responsável).
- **Colunas:** para tabelas, selecione quais colunas exibir e em qual ordem. Os campos padrão são nome, status, responsável e percentual, mas outros podem ser adicionados (ex.: término previsto, sigla da instituição).

Para o componente **Filtros**, escolha quais filtros estarão disponíveis (ex.: responsável, tag).

### Ajustando o layout

Após adicionar cada componente:

- **Redimensione** arrastando as bordas do elemento.
- **Reposicione** arrastando o componente para o local desejado na tela.

---

## Salvando e visualizando

- Clique em **Salvar Alterações** para gravar o dashboard.
- Clique em **Visualizar** para entrar no modo de apresentação. Os controles de configuração ficam ocultos e apenas os dados são exibidos. Ideal para uso em reuniões.
- Para voltar à edição, clique no ícone de **lápis**.

---

## Gerenciando dashboards existentes

Na lista de dashboards, clique na **engrenagem** ao lado de cada item para acessar as opções:

| Ação | Descrição |
|---|---|
| **Abrir** | Visualiza o dashboard (somente leitura) |
| **Editar** | Abre o modo de edição para adicionar ou ajustar componentes |
| **Renomear** | Altera o nome e a descrição do dashboard |
| **Copiar** | Duplica o dashboard para usar como base de uma nova versão |
| **Remover** | Exclui o dashboard permanentemente |

> Ao abrir um dashboard, os dados são consultados em tempo real na plataforma. Dependendo do volume de informações, o carregamento pode levar alguns instantes.
