# Como configurar atalhos de acesso rápido na Plataforma

As páginas são elementos essenciais para a visualização e o acompanhamento dos projetos na Plataforma. Para facilitar a navegação e agilizar o acesso a funcionalidades importantes, é possível criar **atalhos de acesso rápido** tanto para páginas do sistema quanto para links externos.

Neste artigo, você vai aprender como adicionar, editar e controlar o acesso a esses atalhos.

## Introdução

O recurso de acesso rápido permite adicionar atalhos:

* Para páginas da própria Plataforma;
* Para links externos (sites, documentos na nuvem, etc.).

Esses atalhos podem ser visualizados no **desktop** ou no **aplicativo**, e ajudam a otimizar a navegação e a produtividade dos usuários.

## Como adicionar um atalho para página da Plataforma

1. Acesse a **página desejada** dentro da Plataforma.
2. Clique no ícone de **acesso rápido** na barra de navegação.
3. Escolha onde o atalho ficará visível:

   * **Desktop**;
   * **Aplicativo**.
4. Defina a seção do atalho:

   * **Minhas Páginas**: visível apenas para você;
   * **Páginas Padrão**: visível para vários usuários, com controle de acesso configurável.

> **Observação:** Se escolher a seção "Páginas Padrão", lembre-se de configurar o controle de acesso posteriormente.

---


## Como adicionar um link externo como atalho

1. Clique no ícone de **engrenagem** no canto superior direito da tela (ou acesse pelo menu: **Perfil de usuário > Nova Administração > Acesso Rápido**).
2. Escolha o local de exibição: **desktop** ou **aplicativo**.
3. Clique em **Adicionar página externa**.
4. Preencha:

   * **Nome do atalho**;
   * **Endereço (URL)** do link externo.
5. Clique em **Salvar**.

---


## Como editar, personalizar ou excluir um atalho

* Para **editar** um atalho:

  1. Acesse a tela de configuração de acesso rápido.
  2. Clique no ícone de **lápis** ao lado do atalho.

* Para **alterar o ícone**:

  1. Clique em **Mudar ícone**;
  2. Escolha o novo ícone;
  3. Salve.

* Para **renomear o atalho**:

  1. Selecione a opção **Renomear**;
  2. Digite o novo nome e salve.

* Para **excluir** um atalho:

  1. Clique no lápis;
  2. Selecione **Excluir**;
  3. Confirme a ação.

## Como configurar controle de acesso

Caso você tenha adicionado um atalho na seção "Páginas Padrão", será necessário definir quem pode acessá-lo.

### Identificando atalhos sem acesso configurado

* Eles aparecem com uma cor diferente na lista de atalhos.

### Opções de acesso:

1. **Liberar para todos os usuários:**

   * Clique no lápis > **Aplicar a todos os usuários** > Confirmar.

2. **Acessar a tela de configuração:**

   * Clique na **engrenagem** > habilite a opção "Agrupar por grupo de usuários" para ver os níveis de permissão.

3. **Definir acesso por grupo de usuários:**

   * Clique sobre o atalho desejado > selecione os grupos com acesso permitido.
   * O grupo precisa estar previamente cadastrado no sistema.

> **Dica:** Se ainda não souber como cadastrar grupos de usuários, confira o artigo sugerido ao final desta página.

## Como definir um atalho como página inicial

* Para definir um atalho como **sua página inicial**, clique em: **Definir como página inicial**.
* Para **alterar a página inicial de todos os usuários**, clique em: **Redefinir página inicial padrão**.

## Conclusão

Agora você já sabe como criar e configurar atalhos de acesso rápido para páginas da Plataforma ou links externos. Utilize esse recurso para tornar seu dia a dia mais eficiente e personalizado.
