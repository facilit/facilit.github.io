## Introdução

Para facilitar a comunicação entre equipes, agora é possível enviar mensagens diretamente para o WhatsApp do responsável por um projeto, sem sair da Plataforma. A mensagem enviada contém um texto personalizado e o link direto para o projeto, agilizando o processo de revisão e atualização das informações.

---

## Como utilizar a funcionalidade

### 1. Acesse o projeto

* No Gerenciador de Projetos, localize e clique sobre o projeto desejado.

### 2. Localize o ícone do WhatsApp

* Se a funcionalidade estiver ativada, o ícone do WhatsApp aparecerá sobre a foto do responsável pelo projeto.

### 3. Clique no ícone para abrir a mensagem

* Uma janela se abrirá com um texto pré-configurado. Este texto pode ser alterado conforme a necessidade.
* O link do projeto será automaticamente incluído na mensagem.

### 4. Defina a data-limite para resposta

* No mesmo campo, defina até quando o responsável deve responder à solicitação.

### 5. Clique em "Enviar"

* Ao clicar em "Enviar", a Plataforma aciona a API do WhatsApp.
* A aplicação do WhatsApp é aberta (Web ou App) com o telefone do responsável e a mensagem preenchida.

### 6. Confirmação de envio

* Após o envio, a mensagem e o link do projeto ficam registrados na interface da Plataforma.

---

## Como habilitar a funcionalidade

Para que o envio de mensagens via WhatsApp esteja disponível, é necessário que um administrador da Plataforma habilite a funcionalidade. Siga os passos abaixo:

1. Acesse o menu **Administração**.
2. Vá até **Funcionalidades**.
3. Navegue até **Entidade > Outros**.
4. Localize a opção **Envio de mensagem pela página do Painel**.
5. Habilite a funcionalidade no nível desejado.

---

## Observações importantes

* É necessário que o responsável tenha o número de celular cadastrado corretamente no sistema.
* A mensagem é baseada em um modelo configurado previamente no módulo de Notificações.
* Para alterar esse modelo, acesse o artigo **[Configuração de Modelos de Notificação](#)**.

"Olá! Algumas informações no projeto *Revitalização da Praça Central* precisam ser revisadas. Acesse o link abaixo para atualizar os dados até o dia 15/12. [Link do Projeto]"

## Conclusão

O envio de mensagens via WhatsApp direto da Plataforma torna a gestão de projetos mais dinâmica, reduz o tempo de resposta e melhora a integração entre os membros da equipe.

Se tiver dúvidas, consulte também o artigo sobre [Cadastro de Responsáveis](#).
