# Diferença entre Status e Status Informado

Os status são indicadores que mostram o progresso ou a fase atual de um projeto ou ação estratégica na Plataforma. Neste artigo, você vai entender a diferença entre o **status automático** e o **status informado**, como cada um funciona, e de que forma usá-los no acompanhamento das suas iniciativas.

## Introdução

Acompanhar o andamento de projetos é essencial para tomar decisões ágeis e planejar as próximas etapas. Na Plataforma, existem dois tipos de status que ajudam nesse controle:

* **Status automático** (ou simplesmente "status")
* **Status informado**

Vamos entender o papel de cada um.

## O que é o status automático

O status automático é definido automaticamente pelo sistema com base nas **datas previstas e realizadas** da entidade (como objetivos, projetos ou etapas).

### Exemplos:

* Se a data de término previsto já passou e a entidade não foi concluída, o status muda para **Atrasado**.
* Quando você informa uma data de início realizado, o status muda para **Em andamento**.

### Como personalizar os status automáticos:

1. Acesse **Gerenciamento** na barra principal de navegação.
2. Clique em **New Cadastros Auxiliares** 
3. No menu lateral, selecione **Status**.
4. Clique no ícone de engrenagem ao lado do status que deseja editar.
5. Altere o **nome** e a **cor** conforme preferir.
6. Clique em **Salvar**.

## O que é o status informado

O status informado é definido manualmente durante a **criação ou edição** de uma entidade. Ele não está vinculado às datas do projeto ou da ação.

Esse status é útil quando você quer indicar uma situação específica do projeto que não depende de datas.

### Exemplos:

* **Licitação**
* **Orçado**
* **Em análise**
* **Aguardando aprovação**

### Como criar um status informado:

1. Acesse **Gerenciamento > New Cadastros Auxiliares**.
2. No menu lateral, clique em **Status Informado**.
3. Clique em **Adicionar status informado**.
4. Preencha o **nome** e defina uma **cor** para identificação.
5. Clique em **Salvar**.

Para editar um status existente, clique na engrenagem ao lado do nome e selecione **Editar**.

## Como ativar a exibição do status informado

Para que o status informado apareça na página da entidade:

1. Clique no seu nome de usuário no canto superior direito.
2. Selecione **Nova Administração**.
3. Acesse a aba **Funcionalidades**.
4. Marque a opção de exibição de **status informado**.

Depois disso, ele será exibido na página da entidade, próximo às tags.

## Como os status funcionam juntos

O status informado **não substitui** o status automático. Ambos funcionam simultaneamente.

### Exemplo combinado:

Imagine que um projeto está com status informado "Em análise" e que a data de término previsto já passou. O sistema continuará mostrando o status automático como "Atrasado", enquanto o status informado ajuda a contextualizar o motivo do atraso.

Isso oferece uma visão mais completa da situação do projeto.

## Conclusão

Compreender a diferença entre status automático e status informado permite um acompanhamento mais preciso das ações na Plataforma.

Use os dois recursos de forma complementar para manter sua equipe bem informada e facilitar a gestão dos seus projetos.
